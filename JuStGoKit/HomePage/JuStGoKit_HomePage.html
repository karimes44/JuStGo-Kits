JuStGo Kit: Home Page

Introduction

Working with the JuStGo Kit

In this JuStGo kit you will build a Home Page for yourself or for someone you admire. The kit will lead you step-by-step through each task that needs to be completed to build the entire page.

You will find explanations, tips, and resources that help you complete each task. At the end a solution example is given. You can decide for yourself if you want to take over some of the provided example code or stay with your solution.

Note: To see the HTML code, open the solution proposal in your text editor, such as Chrome Caret, or right-click anywhere on the solution proposal page and select View Page Source.

The provided solution example solves the task with very little code so that you can clearly see what is done and how. As a consequence, it may look a bit plain. Let this inspire you to spice it up!

Go wild - make it yours!

Create an HTML placeholder page that...

  • has a body section.
Objectives

Tips and Resources

Solution Proposal

Highlights of added code

First line defines the content as HTML5:

<!DOCTYPE html>
...

The <html> tag encloses the entire page, additional attributes, here lang="en" can give the browser extra information, such that the page in in English.

<!DOCTYPE html>

...
</html>

The <head> tag contains "meta data" and the title to be shown on the browser tab:

<!DOCTYPE html>
<html lang="en">

<!DOCTYPE html>
<html lang="en">

<head>

...

</html>

Task 2: Improve the look of your page

Well, the page is looking a bit bland... Can you fix that?

Objectives
  • Introduction to CSS
  • Basic and common set of CSS properties
    1. Font customization
    2. Color customization
    3. Margin customization
    4. Padding customization
    5. Border customization
    6. Units
Tips and Resources

You can find a good introduction to CSS at W3 School: CSS Tutorial.

You can use CSS to improve the look of your page, for example:

  • Define a left margin for all elements in the <body> tag.
  • Change the font and the font size of the <h1>, <h2>, and the <p> tags.
  • Select different colors for the text and the background color of a tag.
  • Use padding to place the text within the tag element.
  • Add borders to further enhance some tag elements.
  • Redefine the <em> tag to make it look the way you want. You can find here some examples.

Solution Proposal

Highlights of added code

The CSS code is written in the <style> element, which is best placed into the <head> element: 

<head>
...
<style>
</style>
</head>

Each CSS block for styling an HTML element has the general form: 

<style>
TAG {
 PROPERTY1: VALUE1;
 PROPERTY2: VALUE2;
 ...
}
</style>

For example, to change the font for the <h1> heading, you need to use the font-family property, and you can set the text color with the color property. The values sometimes are just a single word, such as the color blue, or a value with a unit, such as setting the font size to 10pt.

Notes: to define the size of the text or the margins you can use the units of pixels px or points pt (see W3Schools: CSS Units).

h1 {
 font-family:  cursive;
 font-size:    20pt;
 color:        blue;
 background-color: pink;
 padding:      20pt; }

The padding property let's you space the text a bit away from the edges of the element - try changing the number to see what happens.

The margin let's you control the spacing between different elements - try changing the number to see what happens.

p {
 font-family:  "Verdana";
 margin-left:  10pt;
 margin-right: 10pt; }

Setting the property font-weight to the value bold creats a bold font.

em {
 font-family:  "Georgia";
 font-weight:  bold;
}

You can also define a border around most HTML elements 

h2 {
 font-family: "Helvetica";
 font-size:    16pt;
 margin:       10pt 60pt 10pt 20pt;
 padding:      10pt;
 background-color: lightgreen;
 border-style: solid;
 border-color: green;
 border-width: 4px;
 border-radius:8px; }
</style>

Task 3: Add pictures and links

Wouldn't the page look even better with some pictures? Can you add some?

... and wouldn't the page be more useful if it had some links to other pages? Can you add them?

Objectives

Introduction to

  • Image tags

Tips and Resources

You can use the image tag to add a picture on to your page.

  • Once you found a picture you like on the web, you can copy the URL (that is the "home address" on the web) of, here, the picture you want to include by right-clicking on the picture and then selecting "Copy image address".
  • Then you move the mouse into your editor window into the src= attribute of your <img> tag and right-click again, this time selecting the option "Paste".
  • ... or you can download the picture onto you computer by right-clicking on the picture and selecting: "Save image as..." - then save it into the same directory as you HTML file, or in a sub-directorty the directory with the HTML file.

A URL for a picture will look "similar" to this:

    https://codenation.org/wp-content/uploads/2021/07/CNVolunteers1-1-800x447.jpg

It will start with https:// or http://, then some long string with some slashes (/) and dosts (.), and it will end often with .jpg or .png, but there are also several other picture formats...

If you saved the file into a sub-directory called images of the directory with the HTML file, then the URL would just be:

    ./images/CNVolunteers1-1-800x447.jpg

You can use the anchor tag to add a link on to your page.

  • To get the URL for a web page you can copy the string from the address bar of your browser, say with Ctrl-C.
  • Then you move the mouse into your editor window into the href= attribute of your <a>and paste the URL with, say Ctrl-V.

A URL for a web page will look "similar" to this:

    https://codenation.org

Task 4: Page layout: columns

The page is getting long, but there is lots of space on the left... can you make better use of that?

Maybe you can make the three sections (the favorite game, things to know, learn more) side by side...

Objectives
  • Controlling the page layout with <div>...</div> and the CSS flex-box property.
  • Targeted CSS element control: the class="..." attribute and the id="..." attribute in HTML and how to use them in CSS.
Tips and Resources

Use the <div> tag to enclose, first each of the sections that you want to see in a side-by-side columnar format, and secondly enclose all these section together in abother <div> tag. Give each tag a descriptive id= or class=.

In the CSS <style> section add to the CSS definition of the outer <div> the display property with the value flex, as well as the justify-content property with the value space-around.

Then style each of the section <style> as you like.

For more information see W3Schools:

  • HTML Block and Inline Elements (<div>)
  • HTML class Attribute
  • HTML id Attribute
  • CSS Selectors
  • CSS justify-content Property
  • W3 School: CSS Tutorial

Solution Proposal

Highlights of added code

Enclose each of the three sections with <div> tags, and then all three with another <div> tag.

The outer <div> tag gets the class attribute "columns" (plural), and each of the three inner <div> tags get class attribute "column" (singular) and other id unique attribute, such as "col1" 

<div class="columns">
 <div class="column" id="col1">
  <p>My favorite games are:</p>
  <ol>
  <li><a href="https://en.wikipedia.org/wiki/Pok%C3%A9mon_Yellow">Pokemon Yellow</a></li>
  <li><a href="https://en.wikipedia.org/wiki/Pok%C3%A9mon_Sun_and_Moon"> Pokemon Sun and Moon</a></li>
  <li><a href="https://en.wikipedia.org/wiki/Pok%C3%A9mon_Legends:_Arceus"> Pokemon Legends: Arceus</a></li>
  </ol>
 </div>

 <div class="column" id="col2">
  <p>Did you know:</p>
  <ul>
   <li>Atsuko Nishida drew me first</li>
   <li>I look a bit like a squirrel</li>
   <li>I am electric</li>
   <li>I can evolve into a Raichu</li>
  </ul>
 </div>

 <div class="column" id="col3">
  <p>Do you want to know more? Check out my
  <a href= "https://en.wikipedia.org/wiki/Pikachu#Appearances">Wikipedia page</a>.
  </p>
 </div>
</div>

in the CSS <style> section the CSS definition for the columns class is (remember for classes the CSS definition starts with a dot!):

.columns {
 display:      flex;
 justify-content: space-around; }

Now you can define the style all three three columns have in common:

 .column {
 width:        30%;
 border-style: dashed;
 border-width: 4px;
 border-radius:8px; }

Use the hash mark # to define the CSS properties per ID. Use this to define the style properties that are different between the three columns:

#col1 {
 background-color: aquamarine;
 border-color: cadetblue; }

#col2 {
 background-color: lightblue;
 border-color: blue; }
 
#col3 {
 background-color: lightpink;
 border-color: firebrick; }

Task 5: Add Likes counter

Let's see what the visitors of you page think on it... Can you add a "like" buttons and keep track of how many visitors clicked then?

Note: In this simple version of your page the lines counter will reset when you reload the page. In a "real" web-page when you click the "like" button that information is sent to a remote computer, called a server, which keeps track of it, but that is for much later in a fellowship program - stay tuned.

Objectives
  • First exposure to JavaScript!
  • Embedded scripts
  • Global variables, local variables
  • Basic function
  • Reading the value of an element
  • Writing HTML into an element
Tips and Resources
  • Add a "Like" button the the end of the page. (Here is list of emoji character codes.)
  • Give it an onclick event handle that calls a function, say newLike().
  • Create an HTML tag that you can write the number of likes to and give it an unique id (say "likes").
  • Create a <script> section in which you define a variable that stores the number of likes (say numberOfLikes) as a global variable (that is outside of any function definition).
  • Then write the newLike() function, which increases the value of numberOfLikes whenever you click on the button, i.e., call the newLike() function.
  • The use the document.getElementById("likes").innerHTML method to write back the current number of likes.

Resources at W3Schools:

  • JavaScript Introduction
  • JavaScript Where To
  • JavaScript Statements
  • JavaScript Variables
  • JavaScript Operators
  • JavaScript Functions
  • HTML DOM Document getElementById()
  • HTML DOM Elements

Solution Proposal

Highlights of added code 
<h3>What do you think?</h3>

<p>Do you like the page?
<button id="like" onclick="newLike()">&#128536;</button></p>

<p>Number of likes: <span id="likes">0</span></p>

<script>
let numberOfLikes = 0;
function newLike() {
 numberOfLikes = numberOfLikes + 1;
 document.getElementById("likes").innerHTML = numberOfLikes;
}
</script>

Task 6: Add a comment section

Ok, Likes are pretty good, but what if a visitor wants to give you more feedback? Can you add a section to leave comments?

Note: In this simple version of your page the lines counter will reset when you reload the page. In a "real" web-page when you click the "Submit Comment" button that information is sent to a remote computer, called a server, which keeps track of it, but that is for much later in a fellowship program - stay tuned.

Tips and Resources
  • Add an <input> tag to the end of the page for visitors to type in a comment.
  • Add a "Submit" button with an onclick event handle that calls a function, say getComment().
  • Create an HTML tag that you can write the number of likes to and give it an unique id (say "comments").
  • Create a <script> section in which you define a variable that stores the all comments (say comStr) as a global variable (that is outside of any function definition).
  • Then write the getComment() function, which concatinates the current string of comments with any new ones whenever you click on the button, i.e., call the getComment() function.
  • You may want to add paragraph or line break tags to separte the different comments.
  • The use the document.getElementById("comments").innerHTML method to write back the current string of comments.

Resources at W3Schools: (same for the previous task.)

Solution Proposal

Highlights of added code 
<h3>Leave a Comment</h3>
<input id="enterComment" placeholder="Enter a comment here"> 
<button id="postComment" onclick="getComment()">Submit</button>

<h4>List of comments</h4>
<div id="comments">Be the first to leave a comment!</div>

<script>
let comStr = "";
function getComment() {
 let newCom = document.getElementById("enterComment").value;
 comStr = comStr + "<p>"+ newCom +"</p>";
 alert(comStr);
 document.getElementById("comments").innerHTML = comStr;
}
</script>
